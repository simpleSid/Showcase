# Project Instructions

- Keep HTML, JavaScript, and CSS in separate files.
- Avoid inline `<script>` and `<style>` blocks unless there is a strong technical reason.
- Prefer clear file names that reflect their role, such as `index.html`, `main.js`, and `styles.css`.
- When creating something new, use the design skills in this order: first create the initial version, then apply `high-end-visual-design` to make it feel premium, then apply `web-design-guidelines` and `impeccable` to review and polish it.
